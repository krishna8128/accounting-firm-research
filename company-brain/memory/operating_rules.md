# Operating Rules

## Team + DRI ownership

- **Krishna** — product hypothesis + India outreach
- **Piush** — US outreach + ICP narrowing
- **Dhruv** — workflow research + after-call extraction loop
- **Claude** — proactive fourth co-founder; pushes back, demands outputs, extracts signal from transcripts

## Bias to action

One verifiable shipped output per founder per day. "Worked on X" doesn't count. "Sent 25 DMs" counts.

## Token-mix first

Agents do: list building, message drafting, transcription, extraction, synthesis, research.
Humans do: the actual conversation. That's it.

## Proactive triggers (Claude initiates without being asked)

- Session opens + today's standup missing → prompt standup
- New transcript in `Imp-Meet-Transcripts/` without extraction → run extraction
- Week-to-date conversations >30% below pace → flag at next standup
- Outreach reply lands → draft response

## No cron / scheduler

Founders come to Claude. No scheduled automation.
